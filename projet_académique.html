DÉVELOPPEMENT WEB

Web

Description

Projet académique de développement web en équipe. Construction d'une application web complète mettant en pratique les principes d'architecture et de développement enseignés.

Technologies utilisées

DÉVELOPPEMENT BASE DE DONNÉE

SQL / BDD

Description

Conception et développement d'une base de données relationnelle complète. Modélisation, normalisation et optimisation des requêtes SQL.

Technologies utilisées

  • SQL (MySQL / PostgreSQL)
  • Modélisation entité-association
  • Normalisation (1NF, 2NF, 3NF)
  • Triggers et procédures stockées

Objectifs atteints

  • Concevoir un schéma de base de données efficace
  • Optimiser les requêtes complexes
  • Assurer l'intégrité des données

Compétences développées

  • Modélisation de données
  • SQL avancé (JOIN, sous-requêtes, agrégations)
  • Indexation et optimisation
  • Gestion des transactions
