CRÉATION SITE DE VENTE

E-Commerce

Description

Développement d'une plateforme de vente en ligne. Ce projet professionnel m'a permis de mettre en œuvre les principes du développement web moderne et de l'e-commerce.

Technologies utilisées

  • HTML
  • CSS

Objectifs atteints

  • Implémenter un système de panier et de commande
  • Assurer la compatibilité multi-navigateurs

Compétences développées

  • Design responsive et multi-navigateurs
  • Intégration de fonctionnalités e-commerce
  • Optimisation des performances web

CRÉATION JEUX-VIDÉOS

Game Dev

Description

Conception et développement d'un jeu vidéo dans un contexte professionnel. Ce projet m'a exposé aux techniques de game design, programmation de jeux et optimisation.

Technologies utilisées

  • Thinter
  • Python
  • Système de physique et collisions
  • Assets et graphiques

Objectifs atteints

  • Développer un jeu pour l'économie
  • Créer un interface utilisateur intuitive

Compétences développées

  • Programmation de jeux et logique
  • Animation et graphismes
  • Travail d'équipe multidisciplinaire
